4. Реализовать интеграцию с ЕГАИС для Продавцов.
5. Создать десктоп-версию приложения для Продавцов.
6. Разработать рекламные инструменты для Виноделен (баннеры, рекомендации).
7. Провести денормализацию данных в OrderItems (копировать данные о товаре).
8. Сгруппировать заказы от разных продавцов в UI на экране "Мои заказы".
9. Реализовать встроенного AI-помощника для покупателей, чтобы он мог порекомендовать вино по их запросам, предпочтениям и бюджету.

10. Реализовать функцию поиска вина по штрихкоду/этикетке для покупателя
    - Пользователь фотографирует этикетку.
    - Изображение отправляется в OCR-сервис (Google/Yandex Vision).
    - Распознанный текст отправляется на бэкенд для "умного" поиска по базе данных вин.

11. Реализовать на странице поиска блок с историей поиска с возможностью очистки запросов по одному или всех сразу.

12. **Подготовить архитектуру к мультиязычности (интернационализация):**
    - Интегрировать `flutter_localizations` и `intl`.
    - Провести рефакторинг: вынести все "захардкоженные" строки в `.arb` файлы.
    - Настроить кодогенерацию для ключей локализации.

13. **Реализовать поддержку русского и английского языков:**
    - Создать файлы `app_ru.arb` и `app_en.arb`.
    - Перевести все строки интерфейса на английский язык.

## Система скидок

Реализовать полнофункциональную систему скидок для товаров.

### Требования:
- **Модель `Offer`:** Добавить поля `old_price` (старая цена) и `discount_percentage` (процент скидки, вычисляемый).
- **UI Карточки товара:** Отображать старую зачеркнутую цену, новую цену и плашку с процентом скидки.
- **Сортировка:** Реализовать логику для сортировки "С большими скидками" (`discount_desc`) в `WinesRepository`.
- **Админка:** Предусмотреть возможность для продавцов управлять ценами и скидками.
