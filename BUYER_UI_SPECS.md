# Спецификация интерфейса покупателя WinePool

## Конкурентный анализ

### OZON и Wildberries
- **Главная страница:** Баннеры с акциями, категории товаров, популярные и рекомендуемые товары, персонализированные подборки.
- **Каталог:** Фильтры по цене, бренду, рейтингу, категориям; сортировка по цене, популярности, новизне.
- **Карточка товара:** Фото товара, детальное описание, характеристики, отзывы пользователей, цена и варианты доставки.
- **Корзина и оформление:** Просмотр товаров в корзине, расчет итоговой суммы, выбор адреса доставки, способа оплаты, подтверждение заказа.

### Vivino
- **Главная страница:** Рекомендации вин на основе предпочтений, поиск по фото, популярные вина.
- **Каталог:** Фильтры по типу вина (красное, белое), стране, цене, рейтингу; сортировка по рейтингу и цене.
- **Карточка товара:** Детальная информация о вине (производитель, год, характеристики), пользовательские рейтинги и отзывы, рекомендации похожих вин.
- **Корзина и оформление:** Интеграция с магазинами для покупки, но фокус на оценках, не на полном checkout.

### Delectable
- **Главная страница:** Сообщественные рекомендации, подборки по регионам и стилям, профили виноделен.
- **Каталог:** Фильтры по стране, региону, сорту, цене; акцент на отзывы и рейтинги сообщества.
- **Карточка товара:** Фото, описание, характеристики, детальные отзывы, социальные аспекты (кто оценил).
- **Корзина и оформление:** Похоже на Vivino, с возможностью покупки через партнеров.

**Адаптация для WinePool:** Комбинировать маркетплейс-подход (OZON/WB) с винной спецификой (Vivino/Delectable). Фокус на удобстве поиска и фильтрации вин, детальных характеристиках, отзывах и предложениях от продавцов.

## Основные разделы интерфейса покупателя
- Главная страница
- Каталог товаров (вин)
- Поиск и фильтрация
- Карточка товара
- Корзина
- Оформление заказа
- Профиль покупателя (история заказов, избранное, настройки)

## Детальное описание

### Главная страница
Главная страница должна обеспечивать быстрый доступ к ключевым функциям и персонализированный контент для вовлечения пользователя.

**Блоки:**
- **Баннеры:** Слайдер с акциями, новинками и специальными предложениями (например, "Скидки на французские вина").
- **Подборки товаров:** 
  - "Популярные вина" (на основе продаж)
  - "Рекомендации для вас" (персонализированные на основе истории просмотров/покупок)
  - "Новинки" (последние добавленные вина)
  - "По категориям" (красное, белое, розовое, игристое)
- **Быстрый поиск:** Поле поиска с автодополнением по названию вина, винодельне или сорту.
- **Навигация:** Кнопки/ссылки на основные разделы (Каталог, Корзина, Профиль).

### Каталог и фильтрация
Каталог представляет собой список вин в виде сетки или списка с возможностью фильтрации и сортировки для удобного поиска.

**Фильтры:**
- Цвет вина: красное, белое, розовое, игристое, десертное
- Содержание сахара: сухое, полусухое, полусладкое, сладкое
- Страна производства
- Регион производства
- Сорт винограда (например, Каберне Совиньон, Шардоне)
- Диапазон цены
- Рейтинг пользователей (звезды)
- Год урожая
- Объем бутылки (0.75л, 1.5л и т.д.)
- Наличие (в наличии, под заказ)

**Сортировка:**
- По цене (возрастание/убывание)
- По рейтингу (высокий/низкий)
- По популярности (количество просмотров/продаж)
- По новизне (дата добавления)

### Карточка товара (вина)
Карточка товара должна предоставлять всю необходимую информацию для принятия решения о покупке, с акцентом на винные характеристики.

**Информация в порядке отображения:**
1. **Фото:** Основное фото бутылки, дополнительные фото (этикетка, виноградник и т.д.)
2. **Название вина**
3. **Винодельня/производитель**
4. **Характеристики:** Год урожая, объем, процент алкоголя, цвет, содержание сахара, страна/регион, сорт винограда, температура подачи, сочетания с едой
5. **Рейтинг:** Средний балл (звезды), количество оценок
6. **Отзывы:** Краткий список последних отзывов, ссылка на полный список
7. **Предложения продавцов:** Список продавцов с ценами, условиями доставки, наличием (например, "Доставка завтра, цена 1500 руб.")
8. **Действия:** Кнопки "Добавить в корзину", "Добавить в избранное", "Поделиться"

### Корзина и оформление заказа
Процесс покупки должен быть простым и интуитивным, с пошаговым руководством.

**Шаги оформления заказа:**
1. **Просмотр корзины:** Список добавленных товаров с возможностью изменения количества, удаления, просмотра итоговой суммы (с учетом доставки и скидок).
2. **Выбор адреса доставки:** Ввод или выбор сохраненного адреса, расчет стоимости доставки.
3. **Выбор способа оплаты:** Опции (банковская карта, наличными при получении, электронные кошельки), ввод данных карты если необходимо.
4. **Подтверждение заказа:** Обзор всех данных (товары, адрес, оплата), итоговая сумма, кнопка "Оформить заказ". После подтверждения - страница с номером заказа и статусом.
